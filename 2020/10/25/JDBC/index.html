JDBC

JDBC（Java Database Connectivity），指Java数据库连接。是Java语言中用来规范客户端程序如何来访问数据库的应用程序接口，提供了诸如查询和更新数据库中数据的方法。

快速入门

环境：安装mysql，注册用户，创建数据库

代码实现：

JDBC访问数据库的步骤：

  1. 注册和加载驱动（可省略）
  2. 获取连接
  3. Connection获取Statement对象
  4. 使用Statement对象执行SQL语句
  5. 返回结果集
  6. 释放资源

对象详解

DirverManager：驱动管理对象

功能：

  1. 注册驱动：告诉程序该使用哪个jar包

    📌注：mysql5之后的驱动jar包可以省略注册驱动的步骤

  2. 获取数据库连接
    方法：static Connection getConnection(String url, String user, String password)

    参数：

     1. **url：指定连接的路径**
    语法：jbdc:mysql://ip地址（域名）:端口号/数据库名称
    例如：jdbc:mysql://localhost:3306/db26
    📌注:如果连接的是本机mysql服务器，且mysql服务默认端口是3306，则url可以简写为：jbdc:mysql:///数据库名称
 2. **user：用户名**
 3. **password：密码**

Connection：数据库连接对象

功能：

  1. 获取执行sql的对象
  2. 管理事务

Statement：执行静态SQL的对象

用于执行静态SQL语句并返回其生成的结果的对象

Statement接口中的方法 描述
int executeUpdate(String sql) 执行DML语句（insert、update、delete），DDL（create、alter、drop）语句
参数：sql语句
返回值：执行DML返回影响的行数，可以通过行数判断是否成功。执行DDL无返回
ResultSet executeQuery(String sql) 执行SQL语句（select）语句
参数：sql语句
返回值：查询的结果集对象

ResultSet：结果集对象

结果集对象,封装查询结果

方法 描述
boolean next() 游标向下移动一行，判断当前行是否是最后一行末尾(是否有数据)，返回true/false
getXxx(参数) 获取数据
Xxx：代表数据类型 如：int getInt() ,String getString()
参数：
  1. int：代表列的编号,从1开始 如： getString(1)
  2. String：代表列名称。 如： getDouble(“balance”)

PreparedStatement：执行动态SQL的对象

表示预编译的SQL语句的对象
