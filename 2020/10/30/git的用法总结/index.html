git的用法总结

git是一个开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

上传项目到github

  1. 在要上传的项目文件夹右键打开git（或者使用cd进入项目文件夹）
  2. 输入git init，回车，用来在当前项目的目录中生成本地的git管理库，当前项目会多一个.git的隐藏文件夹
  3. 输入git add.，回车，将项目中所有的文件添加到仓库中。如果想添加特定文件，把.换成对应的文件名
  4. 输入git commit -m "first commit" ，回车，本次提交的注释，双引号中的字符自定义修改（如果时第一次，需要输入邮箱和名字）
  5. 新建一个github仓库，复制仓库url。输入git remote add origin 仓库url，回车
  6. 输入git push -u origin master，上传代码到github仓库，根据提示输入账号密码即可

更新项目

  1. 打开git
  2. git add.
  3. git commit -m "更新信息"
  4. git remote rm origin
  5. git remote add origin 仓库url
  6. git push -u origin master，输入账号密码
