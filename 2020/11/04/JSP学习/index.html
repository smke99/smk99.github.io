JSP学习

Java服务器页面( Java Server Pages)，是一种动态页面开发技术。JSP是在传统的HTML代码中插入Java程序段和JSP标记而形成的拓展名为“ .jsp ”的文件。JSP本身就是一种Servlet，JSP在第一次被访问的时候会被编译为HttpJspPage类（该类是HttpServlet的一个子类）【参考知乎】

JSP页面组成

指令元素

JSP指令用来设置与整个JSP页面相关的属性，不产生任何可见的输出，只是告诉JSP引擎如何处理JSP页面，如网页编码方式和脚本语言。

语法格式：

常用指令：

指令 用途 范例
page 设定JSP整体信息 <%@ page import="java.util.*"%>
include 在JSP内包含其他JSP内容 <%@ include file="leftframe.jsp"%>
taglib 在JSP内使用自定义标签 <%@ taglib prefix="abc" url="taglib.tld"%>
  1. page指令，常见属性：
    • contentType：等同于response.setContentType()
      1. 设置响应体的mime类型以及字符集
      2. 设置当前jsp页面的编码（只能是高级的IDE才能生效，如果使用低级工具，则需要设置pageEncoding属性设置当前页面的字符集）
    • import：导包。
    • errorPage：当前页面发生异常后，会自动跳转到指定的错误页面
    • isErrorPage：标识当前也是是否是错误页面。
      1. true：是，可以使用内置对象exception
      2. false：否。默认值。不可以使用内置对象exception

注释

  1. html注释：

    <!-- -->:只能注释html代码片段

  2. jsp注释：推荐使用

    <%-- --%>：可以注释所有

内置对象

在jsp页面中不需要创建，直接使用的对象。一共有9个：

变量名（对象） 真实类型 作用
pageContext PageContext 当前页面共享数据，还可以获取其他八个内置对象
request HttpServletRequest 一次请求访问的多个资源(转发)
session HttpSession 一次会话的多个请求间
application ServletContext 所有用户间共享数据
response HttpServletResponse 响应对象
page Object 当前页面(Servlet)的对象 this
out JspWriter 输出对象，数据输出到页面上
config ServletConfig Servlet的配置对象
exception Throwable 异常对象

EL表达式

表达式语言（Expression Language ），EL表达式是用${}括起来的脚本，用来更方便的读取对象！EL表达式主要用来读取数据，进行内容的显示！

语法：

JSP默认支持EL表达式，如果要忽略EL表达式

  • 配置page指令中的isELIgnited = "true"，则不会解析本页EL表达式。
  • 如果要忽略单个EL表达式，则在$前加\即可。

运算

算数运算符：+ - * /(div) %(mod)

比较运算符：> < >= <= == !=

逻辑运算符：&&(and) ||(or) !(not)

空运算符：empty，用于判断字符串、集合、数组对象是否为null或者长度是否为0

  • ${empty list}:判断字符串、集合、数组对象是否为null或者长度为0

  • ${not empty str}:表示判断字符串、集合、数组对象是否不为null 并且 长度>0

从域获取值

EL表达式只能从域对象中获取值，语法：

  • ${域名称, 键名}：从指定域中获取指定键的值

四个作用域对象（从小到大）：

作用域 域名称 生命周期
page pageScope 当前页面
request requestScope 一次请求
session sessionScope 一次会话
application applicationScope 整个Web
  • 如果省略域名称，${name}，则表示依次从最小的域中查找是否有该键对应的值，直到找到为止。

从对象获取值

获取对象、LIst、Map的值

  1. 对象：${域名称.键名.属性名}

    本质上会去调用对象的getter方法

  2. List集合：${域名称.键名[索引]}

    越界会输出空字符串，不会报错

  3. Map集合：

  • ${域名称.键名.key名称}
  • ${域名称.键名[“key名称”]}

隐式对象

el表达式中有11个隐式对象

  • pageContext：获取jsp其他八个内置对象

  • ${pageContext.request.contextPath}：动态获取虚拟目录

JSTL

JSP标准标签库（ JSP Standard Tag Library），JSTL作为最基本的标签库，提供了一系列的JSP标签，实现了基本的功能：集合的遍历、数据的输出、字符串的处理、数据的格式化等等。

使用步骤

  1. 导入相关jar包

  2. 引入标签库，使用taglib指令：

    <%@ taglib %>

  3. 使用标签

常用标签

  1. if ：条件判断。属性：

    • test【必须属性】，接收boolean表达式。如果为true，显示标签体；如果为false，不显示标签体。

    • var：test中判断的结果值的存储变量

  2. choose：类似于if/else。它会与<c:when>及<c:otherwise>标签使用

  3. ⭐foreach：迭代标签

    • 使用循环变量，属性：

      • var：临时变量
      • varStatus：设定变量名，用于存放集合中元素的信息
        • index：容器中元素索引，从0开始
        • count：循环次数，从1开始
      • begin：开始值
      • end：结束值
      • step：步长

    • 循环遍历集合，类似for( Object o : 集合 )，属性：

      • items：容器变量
      • var：设定变量名，用于存储从集合中取出的元素
